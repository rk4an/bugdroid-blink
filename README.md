## Prerequisite ##

* Arduino Nano with ATMEGA328P
* Bluetooth HC-06
* 2 green LEDs
* 2 resitors 220 Ohm
* A bugdroid
* [Bluetooth SPP Pro apps][1]

## Installation ##

* Led right on pin D3
* Led left on pin D4
* HC-06 TX on D6 and RX on D7


  [1]: https://play.google.com/store/apps/details?id=mobi.dzs.android.BLE_SPP_PRO&hl=fr_FR
