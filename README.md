## Prerequisite ##

* [Arduino Nano with ATMEGA328P][2]
* [Bluetooth HC-06][3]
* 2 green LEDs
* 2 resitors 220 Ohm
* A bugdroid
* [Bluetooth SPP Pro apps][1]

## Installation ##

* Led right on pin D3
* Led left on pin D4
* HC-06 TX on D6 and RX on D7
